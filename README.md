# quiethelp-back
* * * * * * * * * * * * * * * * * * * * * * * * * * * * *
descripción

Backend del proyecto QuietHelp.
Se encarga de la lógica del sistema y de exponer una API REST para comunicarse con el frontend desarrollado en Flutter.


* * * - - - - - - - - - - - - - - - - - - - - - - - * * *

tecnologías

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Base de datos: por definir
- Git / GitHub


* * * - - - - - - - - - - - - - - - - - - - - - - - * * *

estructura básica
- controller: endpoints REST
- service: lógica del backend
- repository: acceso a datos
- model: entidades
- resources: configuración

* * * - - - - - - - - - - - - - - - - - - - - - - - * * *

ejecución

- El proyecto se ejecuta como una aplicación Spring Boot.
- Por defecto se levanta en:
    http://localhost:8080
